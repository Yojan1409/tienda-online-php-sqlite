# Tienda Online con PHP y SQLite

Aplicación web desarrollada en PHP que simula un sistema de tienda con gestión de pedidos y base de datos.

## Funcionalidades:
- Búsqueda de productos
- Procesamiento de pedidos
- Conexión a base de datos SQLite

## Tecnologías:
- PHP
- SQLite
- HTML / CSS

## Base de datos:
- Archivo: senati_market.db
